Modular Web Scraping Spider with Python

Modules

- Initialization
	- Imports
	- Browser selection

- Scrape module
	- Has to be written separately for each website

- Multi threading

- (Tkinter) GUI

	- Settings
	- Displays and manipulates data

- File handling module
	- Easy saving to file

Features:
